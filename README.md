# Devoir maison 2025/2026 - Introduction à la programmation orientée objet, à la documentation et aux tests

Ce dépôt contient le *template* pour le devoir maison du cours d'introduction à la programmation orientée objet, à la documentation et aux tests de l'année scolaire 2025/2026.

Comme vous l'avez fait pour les travaux pratiques, vous êtes invité•e•s à initialiser votre dépôt à partir de celui-ci en cliquant sur *Use this template* (en haut à droite, disponible après s'être connecté), en utilisant le même nom pour votre dépôt et **en choisissant la visibilité privée**.
Si vous travaillez à deux, un seul dépôt doit être créé (par une seule personne donc) et la personne ayant créé le dépôt doit ajouter l'autre personne en tant que collaboratrice (*Settings* > *Collaborators* > *Add people* et rentrer le nom d'utilisateur GitHub de l'autre personne).
Cette dernière recevra un mail lui permettant d'accepter l'invitation, ce qui lui donnera les droits pour cloner le dépôt privé ainsi que pour *push* sur la version distante.
